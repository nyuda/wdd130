Purpose

This websit will help you to enter the keyward and then bring up all the movies with title, year and movie poster where the movie title includes the keyward. This site will be very helpful and best place to search your movies.

Content

Home page

The home page has the search bar where the user can enter the keyword and "Get Poster" button. Once the user clicked the button, all the relavant movie list will appear as an output below with Title of the movie, Released year and movie poster. API, "http://www.omdbapi.com/"" will be used to search for the movie and receive the movie information by using fetch. Once the user clicks the title, it will taje the user to IMDV page about the movie.

Images for the Home page

Movie details

Page2 will show the detail of the movie that users clicked on the homepage. I would like to use the Movie Datbase API to fetch the details of the movie.

Images for the Page 2

Movie food

This page will show the people's favorite food and drinks while watching the movie. The users will get an idea of what to prepare before the movie from this page and get ready for the movie. Sometimes people run out of idea about the snacks they would like to have while watching movies. This page will help users to provide the ideas and choose the snacks they want.
